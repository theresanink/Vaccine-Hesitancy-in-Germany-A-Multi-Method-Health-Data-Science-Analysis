# Understanding Vaccine Hesitancy in Germany: A Multi-Method Analysis of ISSP 2021 Survey Data


This project analyzes determinants of vaccine hesitancy in Germany using the ISSP 2021 Health Survey. Logistic regression and tree-based machine learning models are applied to examine which sociodemographic and attitudinal factors are associated with hesitancy. Missing data are handled using multiple imputation and model performance is evaluated using ROC curves and confusion matrices. Results highlight the importance of institutional trust, age and information-seeking behavior for explaining vaccine hesitancy.
